WFC - 3 things
===============

Address
 - Uri
 - where to find the service

Binding
 - protocols. http, tcp ie.

Contract
 - What does the service DO.
 - Is implemented first.

Address, Binding, Contract == End Point


Channels 
 - Channels is what the message goes through between client and host

Behavior
 - Behaviors modify message as they flow through a channel stack. 
	- ie. authorization

Client
 - Proxy = A client communicates with a host through a proxy
	- Proxy = empty type that exposes all operations of a service contract
